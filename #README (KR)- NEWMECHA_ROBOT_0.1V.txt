README (한국어) - NEWMECHA_ROBOT_0.1V

시스템 모듈
- MOTOR DRIVE – 로봇 주행 및 이동
- SENSOR_recog – 센서 인식
- motion_track – 장애물 감지 시 동작 제어
- CRC_tel – 서버와 로봇 간 통신 노드
- vision_CAM – 비전 카메라를 통한 사물 인식
- QR_visible – QR 코드 인식 및 좌표 계산

통신 흐름
서버 → ESP32
- 목적지 경로 설정
예시:
- A에서 B로 이동
- A에서 B로 이동하면서 중간에 C를 들름
- B에 도착 후 D를 거쳐 다시 A로 복귀
디지털 맵 예시:

    _______________ B
A -|---------------- C
    |_______________ D
- ESP32는 서버에서 전송된 QR 값을 읽고 이를 실시간 경로로 변환함
- 서버는 로봇으로부터 공유되는 센서 데이터 및 장애물 감지 영상을 수신함

ESP32 → 서버
- 각 경로 분기점에서 ESP32는 QR 코드를 인식하고 해당 값을 서버로 전송
- 장애물이 감지되면 ESP32는 비전 센서와 초음파 센서를 통해 검사 후 실시간 영상을 전송
- ESP32는 로봇의 다양한 센서 데이터를 서버로 전송

이 문서는 NEWMECHA_ROBOT_0.1V 시스템, 모듈 구성, 그리고 서버와 ESP32 간의 통신 워크플로우를 명확하게 설명합니다.
